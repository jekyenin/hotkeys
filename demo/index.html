Hotkeys.js

Библиотека для создания хоткеев на странице. Включает в себя удобное API управления комбинациями.

Попробовать вживую

Уже сейчас можно нажимать комбинации созданные ниже. Чтобы тригернуть комбинацию надо по ней кликнуть, а для удаления кликнуть на крестик. Prevent - это отмена стандартного действия, например для комбинации Ctrl+E выделится адресная строка, если создать комбинацию с prevent, то строка не выделится.

Первые шаги

Чтобы начать работать с библиотекой много не нужно..

Нужно назначить экземпляр объекта Hotkey переменной и добавить обработчики

var hotkeys = new Hotkeys({ "Ctrl+S": function(){ // Различные штуковины, которые сработают при нажатии комбинации }, "onDone": function(){ // Обработка события окончания действий функции-обработчика комбинации клавиш }, });

Комбинации выглядят следующим образом, по разному в общем.

"Ctrl+S" "Ctrl+Shift+E" "Ctrl+Z" "Ctrl+Alt+Shift+A"

Многие комбинации невозможно перехватить, например: закрытие страницы, важные браузерные комбинации, системные комбинации и т.д. Политика безопасности браузеров запрещает отменять стандартные события важных комбинаций. Такие комбинации как обновление, сохранение и т.п. переназначать разрешено.

API

Методы для управления горячими клавишами на странице. Для взаимодействия с горячими клавишами необходимо обращаться через экземпляр.

Для добавления комбинации клавиш на страницу необходимо в методе add указать комбинацию и метод

hotkeys.add("Ctrl+S", function(){ //Различные штуковины, которые сработают при нажатии комбинации }); /* hotkey.add( combination, function[, prevent] ); combination (string - "Ctrl+S") function (function - function(){}) prevent (boolean - true) */

Для вызова обработчика комбинации из кода нужно использовать метод trigger

hotkeys.trigger("Ctrl+S"); /* hotkey.trigger( combination ); combination (string - "Ctrl+S") */

Чтобы удалить обработчик из списка горячих клавиш необходимо использовать метод remove

hotkeys.remove("Ctrl+S"); /* hotkey.remove( combination ); combination (string - "Ctrl+S") */

Чтобы отключить плагин и приостановить его работу нужно воспользоваться методом destroy

hotkeys.destroy();

События

События устанавливаются при создании экземпляра через методы

Метод onCreate срабатывает при инициализации библиотеки

var options = { onCreate : function(){} }

Метод onAdd срабатывает при добавлении горячей клавиши

var options = { onAdd : function(){} }

Метод onDone срабатывает при срабатывании комбинации при нажатии клавиш

var options = { onDone : function(){} }

Метод onChange срабатывает при изменении метода горячей клавиши

var options = { onChange : function(){} }

Метод onRemove срабатывает при удалении горячей клавиши

var options = { onRemove : function(){} }

Метод onTrigger срабатывает при вызове метода горячей клавиши через код

var options = { onTrigger : function(){} }

Метод onRemove срабатывает при удалении горячей клавиши

var options = { onRemove : function(){} }
